# 🔄 flip плеер

> *"...простой плеер"* (с) Автор, который просто хотел слушать VK и SoundCloud в одном месте, но по пути решил натянуть DDD на Flutter.

**Flip** - это минималистичный, но архитектурно взьерошенный аудиоплеер, который соединяет музыку из VK и SoundCloud. Сделан с душой (и слоями), чтобы было удобно и не стыдно показать код.

## ✨ Чо по фичам?

- 🎧 **Единый плейлист**: Треки из VK и SoundCloud в одной очереди (никакого переключения между приложениями).
- 🔍 **Поиск по двум базам**: Ищешь трек — он шарится и по VK, и по SoundCloud.
- 📦 **Офлайн-кэш**: Скачал - слушай без интернета (кешируем как прокси).
- 🧠 **Умная авторизация**: Зашел через VK ID, прицепил SoundCloud OAuth - и полетели.
- 🎨 **Минималистичный UI**: Ничего лишнего, только плеер, плейлисты и настройки.

## 🏗️ Архитектура (для тех, кто шарит)

Проект написан на **Flutter** с использованием **Clean Architecture + DDD**. Внутри все разложено по полочкам, чтобы не бомбило при добавлении новой фичи.
